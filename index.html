1.ការណែនាំអំពីការកំណត់អ៊ីមែលលើកម្មវិធីផ្សេងៗ

A. Google Gmail “Send As”

  1. ចូលទៅក្នុងគណនី Gmail។
  2. ចុច "Settings" → “See all settings” → “Accounts and Import”។
  3. ចុច “Add another email address”។
  4. បញ្ចូលឈ្មោះអ្នក និងអាសយដ្ឋាន៖ demo@your-name.com
  5. បញ្ចូល SMTP Server:
    • SMTP Server: smtp.yourdomain.com
    • Username: demo@your-name.com
    • Password: (ពាក្យសម្ងាត់អ៊ីមែល)
    • Port: 587 | Security: TLS
  6. បញ្ចូលកូដ Verify →

B. Microsoft Outlook

  1. បើកកម្មវិធី Microsoft Outlook។
  2. ចុច "File" → "Add Account"។
  3. បញ្ចូលអាសយដ្ឋានអ៊ីមែល៖ demo@your-name.com
  4. បញ្ចូលពាក្យសម្ងាត់ →
  5. Manual config:
    • Advanced options → “Let me set up my account manually”
    • ជ្រើស IMAP → បញ្ចូល server, port, និង authentication

C. Mozilla Thunderbird

  1. ចូលទៅកាន់ Mozilla Thunderbird ដែលបានដំឡើងរួច។
  2. បង្អួច "Set Up Your Existing Email Address" នឹងបង្ហាញឡើង...
  3. បញ្ចូលព័ត៌មានខាងក្រោម៖
    • ឈ្មោះពេញ (Full Name): ឈ្មោះអ្នក
    • អាសយដ្ឋានអ៊ីមែល៖ demo@your-name.com
    • ពាក្យសម្ងាត់ (Password): ពាក្យសម្ងាត់របស់គណនី
  4. ចុច “Continue” →
  5. បើ Manual config:
    • Incoming: IMAP - imap.yourdomain.com - Port 993 - SSL/TLS
    • Outgoing: SMTP - smtp.yourdomain.com - Port 587 - STARTTLS
    • Authentication: Normal password
  6. ចុច “Re-test” → បន្ទាប់មក “Done”។
